###  🚵 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
    ```sh
    git clone https://modjuska123.github.io/46-grupe-random-hex-color-generator/
2. Install NPM packages
    npm i
    
    or
    npm install
    
3. Run the server
    npm run dev
